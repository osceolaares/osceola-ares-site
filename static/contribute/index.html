Submitting for Review

Step 1: Commit

Save your changes with a descriptive message in the "Source Control" tab. This creates a snapshot of your work.

Step 2: Push

"Push" your committed changes from CodeSpaces up to your forked repository on GitHub.

Step 3: Pull Request

Create a "Pull Request" (PR) on GitHub. This proposes your changes to the main website administrators.

Step 4: Review & Merge

An admin will review your PR. Once approved, it's merged and published automatically to the live site!
